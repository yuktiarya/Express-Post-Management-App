# 📌 Express Post Management App

A simple and clean CRUD application built using Node.js, Express, and EJS.

## 🚀 Features
- Create new posts
- View all posts
- Edit existing posts
- Delete posts
- RESTful routing
- Method override (PATCH & DELETE)
- Clean and responsive UI

## 🛠 Tech Stack
- Node.js
- Express.js
- EJS
- CSS

## 📂 Project Structure
- views/ → EJS templates
- public/ → CSS files
- index.js → main server file

## ▶️ How to Run

bash
npm install
node index.js
